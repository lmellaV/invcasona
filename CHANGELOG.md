# Changelog

Todos los cambios notables de este proyecto serán documentados en este archivo.

El formato sigue las convenciones de [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/).

## [1.0.0] - 2026-03-16

### Added

- Versión inicial del Procesador de Inventario Cocina
- Funcionalidad de carga de archivos CSV de ventas
- Procesamiento y estandarización de datos de inventario
- Cálculo de consumo basado en factores de conversión
- Exportación a Excel del inventario procesado
- Integración con Google Sheets para actualización automática de datos
- Página secundaria: Generador de Pedidos por Proveedor
- Documentación completa del proyecto

### Features principales

- 📊 Procesamiento automático de reportes de ventas
- 📁 Gestión de archivos local con carga en la nube
- 🔄 Integración bidireccional con Google Sheets
- 📋 Generación de pedidos agrupados por proveedor
- ✅ Validación y limpieza de datos automática
